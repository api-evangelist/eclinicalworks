# eClinicalWorks

eClinicalWorks is a cloud-based EHR and practice management platform serving more than 180,000 physicians and 850,000 medical professionals worldwide. The platform provides REST and FHIR-compliant APIs for accessing patient data, appointments, clinical notes, billing, and health information exchange.

**Developer Portals:**

- Provider-centric and bulk FHIR APIs: [fhir.eclinicalworks.com](https://fhir.eclinicalworks.com/ecwopendev/)
- Patient-centric APIs (healow): [connect4.healow.com](https://connect4.healow.com/apps/jsp/dev/signIn.jsp)

**Authentication:** OAuth 2.0 via SMART on FHIR (EHR Launch, Standalone Launch, Backend Services with Private Key JWT, CDS Hooks)

**API Types:** Clinical API, Scheduling API, RPM Vendor API, Bulk Data Export

**Pricing:** FHIR API access is currently free for certified third-party developers. EHR subscriptions are $449-$599/provider/month.
